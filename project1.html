Project Details

This project is created using "MERN Stack" , in which I have used "Reactjs" for frontend development and "Nodejs" for the backend development. The best part of this project is that you can perform the "CRUD(Create, Read, Update, Delete)" operation on your data. The another best functionality of this project(Expenses-Tracker) is that you can filter your data of expenses according to date, you have filled your data.
